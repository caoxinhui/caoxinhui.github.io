webStorm快捷键汇总
<!-- more -->
command + option + L 格式化代码
command + shift + O 跳到某个文件
ctrl + alt + R 运行文件
alt + f12 打开终端
cmd + d 复制当前行
command + option + esc 杀进程

command + / 单行注释

command + option + / 多行注释

/** + enter 方法注释

command + alt + T 用 if...else... 或者 try...catch 包住

alt + / 与 enter 功能相似，代码自动补全

command + R 替换

command + G 查找下一个

shift + F6 重命名

cmd + home 回到顶部

ctrl + tab 切换导航栏文件

cmd+shift+r 批量替换
ctrl+shift+g 批量查找
