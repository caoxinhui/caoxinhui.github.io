command + shift + L 格式化代码
command + shift + O 跳到某个文件
ctrl + alt + R 运行文件

command + option + esc 杀进程

command + / 单行注释

command + option + / 多行注释

/** + enter 方法注释

command + alt + T 用 if...else... 或者 try...catch 包住

alt + / 与 enter 功能相似，代码自动补全

command + R 替换

command + G 查找下一个

shift + F6 重命名
