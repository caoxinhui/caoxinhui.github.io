斐波那契数列规则 ： `F(N) = F(N-1) + F(N-2)` 
斐波那契数列为 1、1、2、3……
<!-- more -->
👇普通解法，复杂度O(2^n)

``` js
function Fibonacci(n) {
    if (n < 1) return 0
    if (n < 3) return 1
    return Fibonacci(n - 1) + Fibonacci(n - 2)
}
```

👇尾递归优化，复杂度O(2^n)

``` js
function Fibonacci(n, sum1 = 1, sum2 = 1) {
    if (n < 3) return sum2
    return Fibonacci(n - 1, sum2, sum1 + sum2)
}
```

👇从左到右依次求出每一项的值，通过顺序计算求第N项即可，算法复杂度O(N)

``` js
function Fibonacci(n) {
    if (n < 1) return 0
    if (n < 3) return 1
    let res = 1
    let pre = 1
    let tmp = 0
    for (let i = 3; i <= n; i++) {
        tmp = res
        res = res + pre
        pre = tmp
    }
    return res
}
```
